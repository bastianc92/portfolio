# 🚀 Portfolio Personal - Sebastian Castañeda Diosa

Portfolio web personal construido desde cero con HTML, CSS y JavaScript puro. Sin frameworks, sin dependencias.

## 🌐 Demo

[Ver en vivo →](https://bastianc92.github.io/portfolio)

## 🛠️ Tecnologías usadas

- HTML5 semántico
- CSS3 (variables, Flexbox, Grid, animaciones)
- JavaScript vanilla (IntersectionObserver, eventos)

## 📁 Estructura

```
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── (imágenes)
```

## 🚀 Cómo correrlo localmente

1. Clona el repositorio
2. Abre `index.html` con Live Server en VS Code
3. ¡Listo!

## ✨ Features

- Diseño responsivo (mobile-first)
- Animaciones con IntersectionObserver
- Navbar con scroll activo
- Barras de habilidades animadas
- Formulario de contacto
- Publicado con GitHub Pages
